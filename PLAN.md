# TRAIDER — AI-трейдер с человеческим апрувом

## Цель проекта

Система автоматически анализирует крипторынок 24/7, находит торговые возможности
и готовит сделку "под ключ". Человек получает уведомление в Telegram, видит полную
картину и нажимает "Подтвердить" — сделка исполняется на бирже автоматически.

**Принцип:** AI делает всю работу, человек принимает финальное решение.

## Полный флоу работы системы

```
Каждые 15 минут:

1. collector.py      → скачивает свежие свечи BTC/ETH/SOL с Binance
2. indicators.py     → считает RSI, MACD, BB, EMA, объём
3. news.py           → берёт последние новости по активу
4. sentiment.py      → FinBERT оценивает тональность новостей (-1 до +1)
5. predictor.py      → XGBoost выдаёт вероятность роста (например 71%)
6. generator.py      → если вероятность > 65% → формируем сигнал
7. risk.py           → считаем размер позиции, SL, TP исходя из баланса
8. formatter.py      → собираем карточку сделки
9. notifications.py  → отправляем в Telegram

Пользователь видит карточку:

  🔔 Сигнал: BTC/USDT LONG
  ━━━━━━━━━━━━━━━━━━━━━━━━
  📈 Вход:        $84,200
  🛑 Stop Loss:   $82,500  (-2%)
  🎯 Take Profit: $88,000  (+4.5%)
  💰 Размер:      $500 (5% депозита)
  📊 R/R:         1 : 2.25
  🤖 Уверенность: 71%
  📰 Сентимент:   Позитивный (+0.68)
  ⚙️ Причина:     RSI отскок от 32, MACD пересечение вверх,
                  объём +38% от среднего

  ⏳ Сигнал активен 10 минут

  [✅ Войти]  [❌ Пропустить]  [✏️ Изменить]

Нажал ✅ →
  order_manager.py     → выставляет ордер на Binance
  position_tracker.py  → следит за ценой
  При закрытии по TP:  → уведомление о прибыли
  При закрытии по SL:  → уведомление об убытке
```

## Риск-менеджмент (встроен в систему)

- Максимум **5% депозита** на одну сделку
- Максимум **3 открытые позиции** одновременно
- Обязательный **Stop Loss** на каждой сделке
- Минимальный **R/R = 1:2** (прибыль минимум в 2 раза больше риска)
- Таймаут сигнала — **10 минут**, после чего сигнал сгорает
- Если не апрувнул — сделка не открывается

## Зачем собирать статистику

- Видим при каких условиях модель ошибается
- Переобучаем модель раз в месяц на реальных сделках
- Если ты отклонил сигнал, а он сработал бы — система это запоминает
- Оптимизируем порог уверенности (65% → 70% если нужно)

## Возможное улучшение — гибрид с Claude AI

После того как система докажет свою работу на Testnet, можно добавить Claude API
как второй фильтр поверх XGBoost. Схема:

```
XGBoost считает сигнал (бесплатно)
    ↓
Если уверенность > 65%:
    Claude получает данные и даёт финальную оценку (1 запрос ~$0.03)
    ↓
Claude: "Подтверждаю / Отклоняю + объяснение на русском"
    ↓
Telegram-карточка с аналитикой от Claude
```

Запросы только по реальным сигналам (5-10 в день), не каждые 15 минут.
Ориентировочная цена: $5-15/месяц — окупается с первой сделки.

Карточка сделки с Claude будет выглядеть так:

```
🔔 Сигнал: BTC/USDT LONG

📈 Вход:        $84,200
🛑 Stop Loss:   $82,500
🎯 Take Profit: $88,000
🤖 Уверенность: 71%

🧠 Анализ Claude:
"Технически ситуация выглядит сильно — RSI отскочил от зоны
перепроданности, MACD формирует бычье пересечение. Объём
подтверждает интерес покупателей. Новостной фон нейтральный.
Риск — возможное давление со стороны $85k уровня сопротивления.
В целом сигнал качественный, соотношение риск/прибыль привлекательное."
```
