# TRAIDER

AI-трейдер с человеческим апрувом. Система анализирует крипторынок 24/7,
находит торговые возможности и отправляет сигнал в Telegram. Ты нажимаешь
"Войти" — сделка исполняется автоматически на Binance.

**Принцип:** AI делает всю работу, человек принимает финальное решение.

---

## Как это работает

```
Каждые N минут (таймфрейм на выбор):

Binance (реальные данные) → Индикаторы → XGBoost модель → Сигнал
                                                               ↓
                                              Telegram: карточка сделки
                                                               ↓
                                              Ты: ✅ Войти / ❌ Пропустить
                                                               ↓
                                              Binance: set_leverage → ордер + SL + TP
                                                               ↓
                                              Telegram: результат сделки
```

## Быстрый старт

### 1. Получить ключи

- **Binance API** — [binance.com/en/my/settings/api-management](https://www.binance.com/en/my/settings/api-management)
  Разрешения: ✅ Enable Reading + ✅ Enable Futures
- **Telegram Bot Token** — написать @BotFather → `/newbot`
- **Telegram Chat ID** — написать @userinfobot

### 2. Настроить окружение

```bash
cp .env.example .env
```

Заполнить `.env`:

```env
TELEGRAM_BOT_TOKEN=токен_бота
TELEGRAM_WHITELIST=123456789,987654321   # Telegram ID через запятую
ENCRYPTION_KEY=...                        # python -c "from cryptography.fernet import Fernet; print(Fernet.generate_key().decode())"
```

Binance API ключи вводятся прямо в боте через `/start` — они хранятся зашифрованно в БД.

### 3. Запустить

```bash
docker compose up --build
```

Всё происходит автоматически:
1. Поднимается PostgreSQL
2. Запускается trainer — обучает модели для всех таймфреймов (1m, 3m, 5m, 15m)
   - Если модель актуальна (< 30 дней) → пропускает, стартует мгновенно
   - Если модель старше 30 дней → переобучает на свежих данных
3. Запускается бот — каждый пользователь настраивает таймфрейм и параметры в настройках

## Управление размером позиции

Два режима — переключаются в настройках:

**Фиксированная сумма** (`$20 / $50 / $100 / $200`)
Бот всегда заходит ровно на эту сумму в USDT независимо от баланса.
Пример: $20 + плечо 5× → notional $100 → qty ≈ 0.00118 BTC ✓

**Процент от баланса** (`1% / 2% / 5% / 10%`)
Классический риск-менеджмент. Размер позиции = % баланса / расстояние до SL.

> ⚠️ **Минимальный ордер Binance Futures:** BTC/USDT = 0.001 BTC (~$85 notional).
> При $20 маржи нужно плечо минимум 5×. Бот автоматически поднимает до минимального лота.

## Как работает обучение моделей

- При старте `trainer` проверяет наличие моделей для каждого таймфрейма
- Скачивает **2 года** реальных данных с Binance (публично, без ключей)
- Обучает **XGBoost** на 30+ фичах: RSI, MACD, Bollinger, EMA, ATR, объём, ценовые ратио
- Модели хранятся в `models/saved/` (не попадают в git)
- Переобучение возможно: из настроек бота кнопкой "🔄 Переобучить"

## Мульти-пользователь

- Доступ по whitelist: `TELEGRAM_WHITELIST=id1,id2` в `.env`
- Каждый пользователь вводит свои Binance API ключи через бота
- Ключи хранятся **зашифрованно** (Fernet) в PostgreSQL
- Каждый пользователь имеет **независимые настройки**: таймфрейм, плечо, размер позиции
- Сигналы и сделки привязаны к конкретному пользователю

## Риск-менеджмент (встроен)

- **Stop Loss** на каждой сделке — рассчитывается по ATR (1.5×)
- **Минимальный R/R = 1:2** — прибыль минимум в 2 раза больше риска
- **Таймаут сигнала** — не апрувнул вовремя → сигнал сгорает
- **Максимум открытых позиций** — не открываем новое пока не закрыто старое
- **Автокоррекция лота** — если расчётный объём ниже минимума биржи, поднимается автоматически
